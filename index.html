🔄 앱 내 업데이트 가이드

친필 앱은 기능 개선을 위해 자체적으로 최신 버전을 다운로드합니다. 이 과정에서 한 번의 권한 설정이 필요합니다.

1. 업데이트 팝업 확인

앱 실행 시 새로운 버전이 감지되면 '업데이트' 버튼을 눌러주세요. 자동으로 다운로드가 시작됩니다.

2. 친필 앱의 설치 권한 허용

다운로드 완료 후 '보안상의 이유로...' 팝업이 뜨면 '설정'을 누르고, '친필' 앱에 대한 '이 출처 허용'을 활성화한 뒤 뒤로가기를 눌러 설치를 완료해 주세요.
